# 📊 MuraBoard

MuraBoard is a **work-in-progress Job Search CRM / tracker application** designed to help manage job applications, companies, and recruitment progress in one place.

The project is being built as a full-stack application to strengthen backend, frontend, and system design skills while simulating a real-world product.

---

## ⚠️ Status

🚧 **Work in Progress (WIP)**

This project is actively being developed.  
Core backend functionality is already implemented, while the frontend and advanced features are still under development.

---

## 🧱 Tech Stack

### Backend
- Node.js
- Express.js
- TypeScript (NodeNext module system)
- Prisma ORM
- PostgreSQL
- Docker

### Frontend
- React
- Vite
- TypeScript

### Infrastructure
- Docker Compose
- PostgreSQL containerized database

---

## 🧠 Core Concept

MuraBoard is designed to manage:

- Companies
- Job offers
- Job applications
- Application statuses and progress tracking

The goal is to centralize the entire job search process into a single structured system.


## 🐳 Running the project

> Requires Docker & Node.js installed

```bash
docker compose up --build
```

Backend runs on:

http://localhost:3001

Frontend runs on:

http://localhost:5173
